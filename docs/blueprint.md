# **App Name**: Pathway Ledger

## Core Features:

- Inventory Navigation: Easily switch between 'Resentments', 'Fears', and 'Harms' inventory worksheets using a tabbed interface.
- Structured Entry and Editing: Input and revise personal reflections within clearly defined, structured tables, pre-populated with guiding examples.
- Secure Local Persistence: Automatically save all inventory progress directly within the user's browser, ensuring privacy and continuity of work.
- Worksheet Data Export (CSV): Download current inventory data as a CSV file, easily importable into spreadsheet applications like Google Sheets or Excel.
- Worksheet Data Export (TSV for Google Sheets): Copy current inventory data to clipboard as a TSV string for direct pasting into Google Sheets, preserving tabular structure.
- Reflective AI Tool: An AI tool that generates clarifying questions or prompts based on user entries to encourage deeper self-reflection and aid in identifying personal responsibility or corrective actions.
- Dynamic Row Management: Add or remove individual rows within any inventory category to efficiently manage and organize entries.

## Style Guidelines:

- Primary color: A deep, trusting blue (#0066E5) evokes clarity, honesty, and purpose.
- Background color: A clean, desaturated cool gray (#EEF1F4), offering a serene canvas for focused work and reducing visual fatigue.
- Accent color: A vibrant and uplifting cyan (#58D6E6) used judiciously for calls-to-action and important interactive elements to guide the user.
- Body and headline font: 'Inter' (sans-serif) for its outstanding readability, modern aesthetic, and neutral, objective presentation essential for self-reflection.
- Utilize a consistent set of minimalist, outline-style icons from a modern library like Lucide, ensuring clarity and unobtrusive visual guidance.
- A clean, responsive single-page layout, structured with a prominent header, intuitive tab navigation for different inventory types, and content confined to a comfortable reading width.
- Incorporate subtle, functional animations for feedback, such as smooth transitions for tab changes, row modifications, and notification messages, enhancing interactivity without distracting the user.
